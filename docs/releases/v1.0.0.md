# Release Notes - v1.0.0

Release date: 2026-07-10

## Summary

`v1.0.0` is the first portfolio-ready release of Task Management System. It includes a full stack task management application with authentication, task ownership, SQL Server persistence, Docker infrastructure, automated tests, coverage reports, and GitHub Actions CI.

This release is intended to be shared with recruiters, technical interviewers, and engineering reviewers.

## Highlights

- Full stack monorepo with React, TypeScript, ASP.NET Core 8, and SQL Server.
- JWT authentication with login and registration flows.
- Task CRUD protected by authenticated user ownership.
- Docker Compose environment for local full stack execution.
- Frontend test coverage above 89%.
- Backend coverage above 83%.
- CI workflow for lint, format, build, tests, coverage, and artifacts.
- Dependabot updates for npm, NuGet, and GitHub Actions.
- Professional README, architecture docs, Mermaid diagrams, changelog, and MIT license.

## Technical Scope

### Frontend

- React SPA with feature-based structure.
- Login and registration pages.
- Protected route handling.
- Task list and task form workflows.
- Typed API clients.
- Vitest and React Testing Library coverage.

### Backend

- ASP.NET Core 8 Web API.
- EF Core persistence.
- Repository pattern.
- Service layer.
- DTO-based contracts.
- JWT bearer authentication.
- Swagger/OpenAPI.
- Integration tests for auth, users, health, and task ownership.

### Infrastructure

- Docker Compose services for frontend, backend, and SQL Server.
- SQL Server persistent volume.
- Health checks.
- Environment variable based configuration.

### DevOps

- GitHub Actions CI on push and pull request.
- Frontend and backend coverage artifacts.
- Backend test result artifacts.
- Dependabot weekly updates.

## Known Limitations

- No production deployment target is included yet.
- Coverage badge provider is not connected yet.
- End-to-end browser tests are not included yet.
- Refresh tokens are not implemented.
- Task filtering, pagination, and sorting are planned for a future release.
